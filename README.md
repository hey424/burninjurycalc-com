# burninjurycalc.com

Burn injury settlement calculator with educational content covering burn severity classifications (TBSA, depth), causes (thermal, chemical, electrical, scald, friction), treatment costs, life-care planning, and product liability vs. premises liability frameworks.

Editorial review: Cosmo Marchetti (Editor-in-Chief, Burn Injury Practice) + Iris Bowen (Contributing Writer, Medical Documentation & Burn Care).
